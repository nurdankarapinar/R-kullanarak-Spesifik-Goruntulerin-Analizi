# R-kullanarak-Spesifik-Goruntulerin-Analizi
Amaç

Bu çalışmada amaçlanan Güneş leke görüntüsünün EBImage paketi kullanarak Güneş leke görüntüsünün çeşitli filtrelerle incelenmesidir.

Giriş

Güneş lekesi,tam gölge(umbra) ve yarı gölge(penumbra) kısımlarından oluşur.Bizim incelediğimiz Güneş lekesi görüntünün(image1) kompleks leke sınıfına girmektedir.Bu çalışmada image1 görüntüsü kullanıldı.

Çalışma aşamaları

İlk aşamada görüntünün R diline uygun açılması için görüntü okundu.

1-Çalışmada EBImage paketinin;
2-Laplecian Fılter(2D Convolution Filter)
3-Blur Fılter(Low-pass Gaussian filter)
4-Median Fılter
5-Grayscale Fılter
6-Maskeleme ,fonksiyonları kullanıldı.
